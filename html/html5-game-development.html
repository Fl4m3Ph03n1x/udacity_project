Make your own HTML5 Game! Learn all the little detailt about JavaScript and HTML5 that help you make a game !

In this course you will be guided by Google's developers and hopefully you will reach the end of the course knowing how to make a web application that is a game.

This course was a true challenge and I have yet completed it with all the merits and distinctions possible. Honestly, this course skips a lot of steps and introduces a ton of advanced concepts that I would not recommend to anyone, unless that anyone has some external help.

This course also teaches you how to use Chrome's developer tools, but even that is not enough, as you will eventually need a good environment to develop advanced scripts with JavaScript.

It is still a very interesting course with an awesome project, and for those who complete it, it gives them a cool game to share!
